Game rules:

The rules of the game are very simple. Just click on the same color ball with two or more connected to eliminate the connected group ball and get the score. There are two modes for the player to choose. The 'Free Mode' can select the width and height of the number of balls and the number of balls, and can also establish the best record of each type. 'Challenge Mode' has a total of 6 levels, as long as the remaining number of balls meets the customs clearance conditions, and can enter the next level, but there will be a variety of unexpected events in the transit to affect the changes in the disk to disrupt the player's wishful thinking, so increase The uncertainty of the clearance, this mode has the same best record function. Although the rules of the game are very simple, and more points can be scored in a single elimination, it is not easy to get a high score. If a choice is wrong, you may not get a high score or a pass, so you need to get a proud score. Players need to be careful.
