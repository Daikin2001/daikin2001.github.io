BallBall ver. 1.01
Chen Program Study (2003)

Screenshots


Game rules:

This is a game that contains puzzles and luck. At the beginning of the game, there will be 5 balls in the first place. The player selects the ball that he wants to move. Just connect 5 or more balls of the same color into one line (vertical, horizontal, diagonal, etc.), and the ball can be eliminated. But every time you move, if you can't eliminate the ball of any color, you will add 3 more balls on the face. When the space on the panel is filled with balls and can no longer be moved, the game ends.

System Requirements:

  • OS: Windows 98SE/2000/ME/XP
  • Hard disk: 415 KB
  • DirectX version: 7

    • Precaution

    Since the operating environment of the program creation system is Windows 98SE, it is executed on my own machine, but it is not guaranteed to be executed in different system environments such as other computers.
